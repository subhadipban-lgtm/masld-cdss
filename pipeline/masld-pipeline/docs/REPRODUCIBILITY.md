# Reproducibility Guide

This document provides step-by-step instructions for reproducing all results in the manuscript *"Decoding the functional signature of ferroptosis-driven fibrosis with a personalised GNN"*.

## System Requirements

### Hardware
- **RAM**: 32GB minimum (64GB recommended for full WGCNA + GNN pipeline)
- **Storage**: 50GB free disk space (raw GEO data + intermediate files)
- **GPU**: Optional but strongly recommended for GNN training (NVIDIA GPU with ≥8GB VRAM, CUDA 11.8+)
- **CPU**: Multi-core processor (8+ cores recommended for parallel R operations)

### Software
- **OS**: Linux (Ubuntu 22.04+ recommended) or macOS 13+
- **R**: Version 4.2 or higher
- **Python**: Version 3.9 or higher
- **Bioconductor**: Version 3.16 or higher

## Step-by-Step Reproduction

### Step 1: Environment Setup

```bash
# Clone repository
git clone https://github.com/yourusername/masld-drugscope.git
cd masld-drugscope

# Create Python virtual environment
python -m venv venv
source venv/bin/activate
pip install -r envs/requirements_python.txt

# Install R packages
Rscript -e '
if (!require("BiocManager", quietly = TRUE)) install.packages("BiocManager")
bioc_pkgs <- c("GEOquery", "sva", "limma", "edgeR", "dorothea", "viper",
               "clusterProfiler", "org.Hs.eg.db", "ReactomePA", "enrichplot",
               "STRINGdb", "GOplot", "impute", "preprocessCore")
BiocManager::install(bioc_pkgs)
cran_pkgs <- c("tidyverse", "pheatmap", "RColorBrewer", "ggrepel", "ggpubr",
               "corrplot", "msigdbr", "igraph", "ggraph", "patchwork", "WGCNA")
install.packages(cran_pkgs)
'
```

### Step 4: Run the Complete Pipeline

```bash
chmod +x run_pipeline.sh
./run_pipeline.sh
```

**Expected runtime:**
- Stages 1-2 (Data harmonization): ~30 minutes (dominated by GEO download)
- Stages 3-8 (R analysis): ~2-4 hours
- Stage 9 (GNN training): ~30-60 minutes (GPU) or 2-4 hours (CPU)

## Troubleshooting

### Common Issues

1. **GEO download fails**: GEO servers may rate-limit. Wait 10-15 minutes and retry, or manually download from NCBI.

2. **ComBat-seq memory error**: Reduce the number of genes by pre-filtering low-expression genes. Ensure at least 32GB RAM.

3. **PyTorch Geometric installation**: Follow the official installation guide at https://pytorch-geometric.readthedocs.io/en/latest/install/installation.html

4. **GNN training NaN loss**: Reduce learning rate to 0.001 or increase batch size. Verify node features are properly normalized.

5. **STRINGdb connection timeout**: STRING API may be temporarily unavailable. Retry or use a local STRING database.

### Reproducibility Notes

- **Random seeds**: The pipeline uses fixed random seeds (Python: `torch.manual_seed(42)`, R: `set.seed(123)`) for reproducibility.
- **Package versions**: Exact versions are specified in `envs/` directory. Using different versions may produce slightly different results.
- **Hardware differences**: GPU vs CPU training may produce minor numerical differences due to floating-point arithmetic. This is expected and does not affect biological conclusions.
